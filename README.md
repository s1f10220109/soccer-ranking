# ⚽ サッカーランキング管理システム

FIFAランキング形式で友達同士のサッカー試合結果を記録・管理するWebアプリケーションです。

## 機能

### 📋 主な機能
- **チーム登録**: 参加チームを簡単に登録
- **試合記録**: 試合結果（スコア、日付、試合タイプ）を記録
- **自動ランキング更新**: FIFAランキングに似た計算方式で自動的にランキングを更新
- **試合タイプ別の重み付け**:
  - 親善試合: x1.0
  - 予選: x2.5
  - 大陸選手権: x3.0
  - ワールドカップ: x4.0
- **データの永続化**: ブラウザのローカルストレージに自動保存
- **データエクスポート**: JSON形式でデータをダウンロード可能

### 📊 ランキングシステム

FIFAランキングシステムに基づいた計算方式を採用：

1. **基本ポイント**
   - 勝利: 3ポイント
   - 引き分け: 1ポイント
   - 敗北: 0ポイント

2. **対戦相手の強さによる補正**
   - 強いチームに勝つとより多くのポイント獲得
   - 弱いチームに負けると大きくポイント減少

3. **試合タイプによる重み**
   - 重要な試合ほど大きくポイントが変動

## 使い方

### 1. サイトを開く

`index.html`をブラウザで開きます。

```bash
# ファイルをダブルクリック、または
start index.html
```

### 2. チームを登録

1. 「チーム登録」セクションでチーム名を入力
2. 「チームを追加」ボタンをクリック
3. 全てのチームを登録

### 3. 試合を記録

1. ホームチームとアウェイチームを選択
2. スコアを入力
3. 試合タイプを選択
4. 試合日を選択
5. 「試合を記録」ボタンをクリック

### 4. ランキングを確認

- 試合記録後、自動的にランキングが更新されます
- ポイント、勝敗数、得失点差などが表示されます

## データ管理

### データの保存
- 全てのデータはブラウザのローカルストレージに自動保存されます
- ページを閉じても、次回開いた時にデータが復元されます

### データのエクスポート
- 「データをエクスポート」ボタンでJSON形式でダウンロード可能
- バックアップやデータ共有に便利

### データのクリア
- 「全データをクリア」ボタンで全てのデータを削除
- 確認ダイアログが表示されます

## カスタマイズ

### 試合タイプの重みを変更

`app.js`の`MATCH_WEIGHTS`オブジェクトを編集：

```javascript
const MATCH_WEIGHTS = {
    friendly: 1.0,      // 親善試合
    qualifier: 2.5,     // 予選
    continental: 3.0,   // 大陸選手権
    worldcup: 4.0       // ワールドカップ
};
```

### 初期ポイントを変更

`app.js`の`addTeam()`関数内の`points`プロパティを変更：

```javascript
const newTeam = {
    // ...
    points: 1000, // この値を変更
    // ...
};
```

## トラブルシューティング

### データが保存されない
- ブラウザのプライベートモード/シークレットモードを使用していないか確認
- ブラウザの設定でローカルストレージが有効になっているか確認

### ランキングが表示されない
- 最低1つのチームを登録してください
- ブラウザのコンソールでエラーが出ていないか確認
